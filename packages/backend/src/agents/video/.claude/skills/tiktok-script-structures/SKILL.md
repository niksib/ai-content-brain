# TikTok Script Structures

## The Only Metric That Matters

Completion rate. In 2026, TikTok requires 70%+ average completion for the algorithm to push content beyond the initial test batch. Every script decision must serve one goal: keep the viewer watching until the end.

This means:
- No slow intros — the interesting thing happens at second 1
- No dead air or padding — every second must earn its place
- Payoff must be worth the setup — the end must justify the watch

---

## Choosing the Right Structure

Before picking a structure, identify the content type:

| Content Type | Best Structure |
|---|---|
| Personal story with a twist | Story Arc |
| Tutorial / how-to | Step-by-Step |
| Opinion / hot take | Contrarian |
| Mistake + lesson | Failure-to-Insight |
| Quick tip or hack | Quick Value |
| Comparison or test | A vs B |

---

## Structure 1: Story Arc
**Best for:** Personal experiences, behind-the-scenes, moments that happened
**Optimal length:** 45-90 seconds

```
[0-3s] HOOK — most interesting moment from the story, mid-scene
        Spoken: the thing that happened
        On-screen: reinforce with text

[3-15s] SETUP — minimum context needed to understand stakes
         Don't over-explain — trust the viewer

[15-60s] THE STORY — what actually happened
          Keep pace fast — one beat every 5-10 seconds
          Add tension or uncertainty: "I had no idea what was about to happen"

[60-end] RESOLUTION + LESSON — what happened, what you learned
          Must be worth the watch
          
[Last 3s] CTA — one specific action
```

**Retention technique:** At second 15-20, add a "but then" moment — something that changes the direction. This resets attention.

---

## Structure 2: Step-by-Step
**Best for:** Tutorials, processes, how-to content
**Optimal length:** 60-180 seconds

```
[0-3s] HOOK — lead with the result, not the process
        "Here's exactly how I [result] in [time]"
        Show the outcome before showing the steps

[3-10s] CONTEXT — why this matters / who this is for
         Keep short — get to the steps fast

[10-60s] THE STEPS — numbered, one per point
          Speak fast
          On-screen text for each step number
          Each step needs a specific action, not vague advice

[60-end] BONUS TIP or RESULT — reward viewers who watched to the end
          "One thing most people miss..." works well here

[Last 3s] CTA
```

**Retention technique:** At step 3 or halfway through, say "but here's the one most people get wrong" — this resets attention and gives viewers a reason to keep watching.

## Structure 3: Contrarian
**Best for:** Opinion content, hot takes, myth-busting
**Optimal length:** 30-60 seconds

```
[0-3s] HOOK — state the contrarian claim boldly
        "Everyone says X. Here's why that's wrong."
        Must feel like a genuine challenge, not clickbait

[3-15s] ESTABLISH THE COMMON BELIEF — briefly validate why people think this
         "I get it. I used to think this too."

[15-45s] THE ACTUAL ARGUMENT — specific evidence, personal experience, data
           Not just assertion — show why you believe this
           One strong argument beats three weak ones

[45-end] CONCLUSION — what they should do instead
          Practical, specific

[Last 3s] CTA — often a comment CTA: "Do you agree? Let me know."
```

---

## Structure 4: Failure-to-Insight
**Best for:** Mistake stories, lessons learned, things that went wrong
**Optimal length:** 45-90 seconds

```
[0-3s] HOOK — state the failure directly
        "I made a [specific mistake] that cost me [specific cost]."
        Don't tease — be direct about what went wrong

[3-15s] CONTEXT — what I was trying to do, why it seemed like a good idea

[15-50s] WHAT HAPPENED — the specific failure, in real detail
           Don't skip the embarrassing parts — authenticity is the value

[50-end] THE INSIGHT — the actual lesson
          Not generic ("always double-check") but specific
          "What I do now instead is..."

[Last 3s] CTA — comment or save
```

**Why it works:** Failure content has high completion because viewers stay to find out what went wrong. The lesson at the end rewards them.

---

## Structure 5: Quick Value
**Best for:** Single tips, hacks, shortcuts
**Optimal length:** 15-30 seconds

```
[0-3s] HOOK — state exactly what you're about to share
        "One thing I do that [specific benefit]"
        No setup — get straight to it

[3-20s] THE TIP — explain it fully, show it if possible
          One idea only — don't pad with related tips

[20-30s] PROOF or RESULT — show it working or state the specific outcome
          "Since doing this, [specific measurable result]"

[Last 3s] CTA — follow for more of this type of tip
```

---

## Structure 6: A vs B
**Best for:** Comparisons, before/after, testing
**Optimal length:** 30-60 seconds

```
[0-3s] HOOK — set up the comparison with stakes
        "I tested [A] vs [B] for [time]. The results surprised me."

[3-10s] SETUP — what each option is, why the comparison matters

[10-35s] THE COMPARISON — go through each one specifically
          Use on-screen graphics if comparing numbers
          Keep it fair — show real results for both

[35-end] WINNER + REASON — clear answer, specific reason
          "Use [X] when..." / "Use [Y] when..."

[Last 3s] CTA
```

## Universal TikTok Script Rules

### Voice and Delivery
- Speak faster than feels natural — dead air kills retention
- One idea per visual moment — cut before the next thought
- Match energy to content: story = conversational, tutorial = clear and direct
- Speak directly to one person, not a crowd ("you", not "everyone")

### On-Screen Text
- Hook text on screen in first 3 seconds — reinforce the spoken hook
- Step numbers or key terms during tutorial sections
- Captions always — 85% watch without sound
- Text should complement, not duplicate — add context the voice doesn't say

### B-Roll Usage
- Use when voiceover explains something the talking head doesn't show
- Match b-roll cut to the sentence rhythm — cut on the beat of speech
- Avoid stock footage — real footage from the creator's actual situation

### Search Optimization
- TikTok is a search engine — use keywords in spoken words (auto-transcribed)
- First 3-5 seconds should contain the main keyword if searchable content
- Include 3-5 specific hashtags — avoid #fyp, #viral (penalized for irrelevance)
- Caption should include searchable terms naturally

### Length Decision
- If completion rate matters most → go short (15-30s) — easier to hit 70%
- If search ranking matters → go longer (60-90s) — more keyword signals
- Only go 90s+ if every second earns its place — test with shorter version first
