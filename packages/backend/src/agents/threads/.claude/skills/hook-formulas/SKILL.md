---

Hook principles:
- The first line must create tension, friction, or curiosity. Anything that lets someone read it and feel nothing fails.
- Specific over vague. "Today X happened" with real detail beats "I learned a lesson".
- Observation beats instruction. Stating what you noticed beats telling people what to do.
- Never start with "I" as the first word — move the interesting detail forward.

Hooks that don't work (data-backed):
- "How to" openers: 0% chance of top 10% across 1996 viral posts
- Generic wisdom ("Success takes hard work")
- Platform announcements ("New post!", "Just published")
- Introducing what the post will cover ("In this thread I'll show you")
- Questions with obvious answers ("Want more followers?")
- Engagement bait ("Like if you agree")

Quality check before posting:
- Does the first line create tension?
- Is it specific enough that only you could have written it?
- Would it make someone want to react?
- Does the post deliver on what the hook implies?
